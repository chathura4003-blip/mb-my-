# 🚀 Supreme MD WhatsApp Bot

A powerful, high-performance WhatsApp bot built with Baileys. Optimized for video downloads, media processing, and seamless session management on Railway.

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https%3A%2F%2Fgithub.com%2Fchathura4003-blip%2Fmb-my-)

## ✨ Features
- **Smart Downloads:** YouTube, TikTok, and more with quality selection (HD/SD/Audio).
- **Mobile Optimized:** Custom FFmpeg scaling ensures videos play directly on WhatsApp mobile.
- **Session Persistence:** Guaranteed to stay connected even after server restarts.
- **Aesthetic UI:** Premium message formatting and a built-in dashboard.
- **Anti-Link:** Automated group moderation.

## 🛠️ Deployment on Railway (1-Click)

1. Click the **Deploy on Railway** button above.
2. Connect your GitHub repository.
3. Railway will automatically:
   - Prompt you for your `OWNER_NUMBER`.
   - Set up the **Persistent Volume** for your session.
   - Deploy the bot!

## 📱 Connecting the Bot

1. After deployment, go to the **View Logs** tab in Railway.
2. Look for the **QR Code**.
3. Open WhatsApp on your phone -> **Linked Devices** -> **Link a Device**.
4. Scan the QR code shown in the logs.
5. Bot will connect and stay online!

## 📜 Commands
- `.menu` - Show all commands.
- `.ping` - Check bot status.
- `.upload <link>` - Download and send video/audio.
